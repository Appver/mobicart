GST Returns

All registered businesses have to file monthly, quarterly and/or annual GST Returns based on the type of business
A return is a document containing details of income which a taxpayer is required to file with the tax administrative authorities.
This is used by tax authorities to calculate tax liability. Under GST, a registered dealer has to file GST returns that include:

  • Purchases
  • Sales
  • Input tax credit (GST paid on purchases)
